Track The I.S.S.

This project uses APIs to check if the International Space Station (ISS) is overhead and if it is nighttime at the user's location. If both conditions are met, it sends an email every 60 seconds to remind the user to look up at the sky.

Linked-In Bot

This project uses Selenium to automate the application process for a Python Developer job on LinkedIn. It navigates to the job page, signs in, and initiates the easy apply process. The script fills in the required information, including a phone number, and completes the application.

Amazon Price Tracker

This project extracts the price of a specific product on Amazon using web scraping with BeautifulSoup and sends an email alert if the product price is below $150. It utilizes the requests library to fetch the webpage, BeautifulSoup for parsing HTML, and smtplib for email notification.
